# 🍕 Pizza Sales Analytics Dashboard

> **Tools Used:** SQL | Tableau
> **Dataset:** 48,000+ records | $817,000+ Revenue

---

## 📌 Objective

To analyze pizza sales data and identify revenue trends, top-performing products, and peak demand patterns to support business decision-making.

## 🔍 Key Insights

- 💰 **Total Revenue:** $817,000+
- 📦 **Total Orders:** 48,620
- 🍕 **Top Category:** Classic pizzas contribute the highest revenue
- 📅 **Peak Days:** Friday and Saturday evenings
- 📉 **Lowest Sales:** Sunday mornings

---

## 🛠️ SQL Concepts Used

- `JOINS` — combining order and product tables
- `CTEs` — Common Table Expressions for readable queries
- `Window Functions` — ranking products by revenue
- `GROUP BY / ORDER BY` — aggregating sales by category and time
- `SUBQUERIES` — filtering top performers
